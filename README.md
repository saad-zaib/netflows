NetFlow Monitoring & Analysis Pipeline

This project provides a complete, automated NetFlow collection, parsing, and security monitoring pipeline for Linux environments. It is designed for home labs, SOC teams, and cloud servers that need real-time visibility into network traffic and security events.
The solution integrates seamlessly with Elastic Stack (ELK) and other SIEM, turning raw NetFlow packets into actionable analytics and security alerts. The datais being stored in json for now
